# Frame a Candidate

Read and follow the `shape-up:frame` skill to run this framing session.

The candidate is: $ARGUMENTS

If no candidate was provided, ask the user to describe the raw idea or request in their own words before proceeding.

Work through the framing questions one at a time. Do not move on until each question has a real answer. Push back on labels and vague problem statements.

When the Frame is complete, write the concept document to `docs/concepts/[name].md` using the full document structure from the skill. Fill in the Frame section. Leave Solution, Bet, and Build as empty placeholders.

If the `docs/concepts/` directory does not exist, create it.

Close by stating clearly whether this reached Frame Go or needs more investigation, and set the status field in the document accordingly.
